artbox.reconfile.load_matlab_dataset(filename, double=False)[source]¶

Load a Matlab struct

The structs name must be S and it’s format is expected to be:

S.k: [nF, nT]

S.SEM: [nF, nX1, nX2]

S.Cmat: [nC, nX1, nX2]

S.b0: [nX1, nX2]

S.ktime: [1, nT]

S.Gmat: [nF, nX1, nX2, 3]

S.w: [3, 1]

S.regularization_weights: [nX1, nX2]

As well as one of the following two:

S.recondata: [nC, nT]

S.object: [nX1, nX2]

Parameters:
  • filename (str) – File to load.
  • double (bool) – Whether or not everything should be double or single precision.
